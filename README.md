# Gusto

## Project Overview

Gusto is a bold, flavor‑focused landing page for a subscription service delivering rare, small‑batch hot sauces and condiments from around the world. Each bottle arrives with a story—spotlighting regional makers, culinary traditions, and unique heat profiles.

## Features

* **Maker Stories & Videos** – Learn the origin, craft process, and regional culture behind each sauce.
* **Recipe Pairing Cards** – Curated pairings to help users make the most of each month’s flavor.
* **Scoville Scale Tracking** – Track heat levels across your collection and discover your preferred spice zone.
* **"Heat & Flavor" Profile Explorer** – Interactive tool showing the balance of smokiness, sweetness, acidity, and heat.
* **"Sauce Match" Quiz** – Helps users find their ideal flavor experience.
* **Responsive Design** – Built with Bootstrap 5 for smooth viewing on all devices.

## Technologies Used

* **HTML5** – Semantic layout and content structure.
* **CSS3** – Custom styles using the project’s bold color palette.
* **Bootstrap 5** – Grid, components, and layout responsiveness.
* **JavaScript** – Interactive flavor explorer and quiz functionality.
* **Favicon** – A single, perfectly shaped chili pepper icon.

## Setup Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/gusto.git
   ```
2. Navigate into the project directory:

   ```bash
   cd gusto
   ```
3. Open `index.html` in your browser, or run a lightweight server:

   ```bash
   npx serve .
   ```
4. Add hero video, images, and data files to the `assets/` directory.

## Intended Use

Gusto is intended for culinary subscription services, artisanal food brands, and spice enthusiasts looking to tell rich maker stories while delivering unique flavors. The landing page demonstrates the service’s value, engages users through interactive tools, and highlights each month’s featured sauce in a visually striking way.
